Exeter Observer submission guide

If you are considering contributing to Exeter Observer please see our contributors guide.

We use Signal for all our team communications, including story text and photography submission.

Please send messages as below from a smartphone, tablet or computer to this number: .

This number does not receive ordinary mobile network calls or texts.

Text

Please submit text to Exeter Observer as a Signal message and NOT via email or as an attached document, Word or otherwise.

All formatting will be removed when the text is pasted into the Signal edit window so please ensure that each paragraph is separated by a line break.

Hyperlinks will also be removed: please use Markdown link syntax to retain them. It looks like this:

[Example link text](https://example.org)

You can indicate the position of photos or graphics, or leave notes for editors using square brackets:

[photo goes here]

[infographic at https://flourish.studio/examples/ goes here]

[quote from source TBC after interview]

Any typographic elements such as bold or italic text will be determined during editing.

No double spaces, please!

If you would like to submit a news brief or feature outline, please simply use line breaks to separate each subhead from the text that follows:

HOOK

text goes here

VALUE

text goes here

PUBLIC INTEREST

text goes here

etc.

Photos

Please submit photographs to Exeter Observer as Signal attachments and NOT via email or dropbox or similar download or file-sharing services.

Photos should be in jpg, png or tiff format and must be at least 800px x 450px @ 240dpi. Ideally they should be at least 1920px x 1080px @ 350dpi.

Please send original, uncompressed images if possible: editors will crop to the correct aspect ratio and compress the files to appropriate sizes for web and/or print publication.

If you are submitting photographs that you have not taken yourself, you must confirm that the copyright holder has given permission for their use.

Such images must be accompanied by appropriate use permission, license and origin information, such as a link to a web page which clearly indicates these usage rights and which will allow Exeter Observer to appropriately credit the copyright holder.

Images of children under sixteen can only normally be used with the consent of their parent or guardian. If in doubt, do not submit such images.

Videos

Videos can be embedded on the website via the Exeter Observer Vimeo account. (We do not use YouTube.)

If you are not the copyright owner, you must confirm that use permission has been granted in the same way as above.

You can use a free account to create an infographic then send the URL of your project to import.

Alternatively, you can share your account URL (e.g. https://app.flourish.studio/@username) and indicate which project you want to use.

Please see our contributor guide on multimedia collateral & copyright for more information and get in touch if you are unsure about any aspect of the above.
